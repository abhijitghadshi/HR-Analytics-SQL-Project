 # HR Analytics SQL Project 

# Project Description
This project focuses on analyzing HR data using SQL.  
It includes employee information, department mapping, salary records,
and attendance tracking to generate meaningful business insights.

# Technologies Used
- MySQL
- SQL (DDL, DML, Joins, Subqueries)

 # Database Structure
- Departments
- Employees
- Salaries
- Attendance

 # Key SQL Queries
- Employee details with department
- Department-wise average salary
- Highest paid employee
- Employees with high attendance
- Employees earning above average salary

# How to Run
1. Clone the repository
2. Run `create_database.sql`
3. Run `insert_data.sql`
4. Execute queries from the `queries` folder

# Learning Outcomes
- Hands-on experience with SQL joins
- Real-world HR analytics use cases
- Improved data analysis skills
