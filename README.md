# HR Analytics SQL Project 

# Project Overview
This project demonstrates how SQL can be used for HR analytics.
It includes employee details, departments, salary records, and attendance data
to generate meaningful insights.

# Technologies Used
- MySQL
- SQL (DDL, DML, JOINs, Subqueries)

# Database Tables
- Departments
- Employees
- Salaries
- Attendance

# Key Queries
- Employee details with department
- Department-wise average salary
- Highest paid employee
- Attendance-based filtering
- Employees earning above average salary

# How to Run the Project
1. Run `database/create_tables.sql`
2. Run `database/insert_data.sql`
3. Execute queries from `queries/analysis_queries.sql`
