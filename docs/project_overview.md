# HR Analytics SQL Project

This project demonstrates how SQL can be used for HR data analysis.
It covers employee management, salary analysis, attendance tracking,
and department-wise insights.

# Key Concepts Used:
- Relational Database Design
- Primary & Foreign Keys
- SQL JOINs
- GROUP BY and Aggregation
- Subqueries

# Use Cases:
- HR reporting
- Payroll insights
- Employee attendance monitoring
